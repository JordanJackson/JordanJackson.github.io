Dungeon Game

This simple game was made with Unity to show off a Map/Dungeon generation tool I created. Given a width and height of a tile map, the algorithm will place rooms (up to a specified number of rooms or failures), and uses a floodfill algorithm to fill the remaining space with corridors. The algorithm then looks for possible connections such that each room is connected to the main network of rooms. All excess corridors are then removed.
Made with: C#, Unity
